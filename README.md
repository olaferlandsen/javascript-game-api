### Features
* Multiple Person
  * Attack, Defense, Critical, Life, Gold, and many more...
* Inventory
* Buy and Sell
* Compatible with jQuery, Angular, React and Cordova.

### Documentation
This API is full documented and you can see it on folder `docs/`
